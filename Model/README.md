# Model 資料夾說明

本資料夾包含專案中所有機器學習模型的定義與訓練流程程式碼。

## 檔案說明

-   `__init__.py`: 將此資料夾標記為 Python 模組。
-   `gat_model.py`:
    -   定義了 GATv2 (Graph Attention Network v2) 模型的架構。
    -   實現了 GraphMAE 風格的無監督預訓練，用於從圖結構中學習通用的節點表示。
    -   實現了監督式微調流程，使用警示帳戶標籤對預訓練模型進行優化。
    -   包含將訓練好的 GNN 用於提取高階衍生特徵 (如：節點風險機率、重建誤差、鄰居風險等) 的功能，並將這些特徵儲存為 Parquet 檔案供下游模型使用。
-   `xgb_model.py`:
    -   負責執行 XGBoost 模型的訓練與預測。
    -   它會整合 `Preprocess` 模組生成的表格特徵以及 `gat_model.py` 產出的 GNN 衍生特徵。
    -   採用 K-Fold 交叉驗證策略進行穩健的模型訓練與評估。
    -   包含錯誤分析、特徵重要性分析等輔助功能。
    -   最終負責生成 `submission.csv` 提交檔案。ㄋ
